Rendu S4 sur le projet d'axe HELLSAVIOR

Les fonctionnalités de GameFeel implémentées : 

- CameraCentroid avec comme cibles le joueur et la crosshair qui reste fluide
- Un menu principal
- Un menu pour séléctionner les armes avant le début de la partie
- Un menu pause
- Une barre de vie
- Un hud pour voir les armes équipées, laquelle est sélectionnée, combien de balles les armes ont et une animation de cette ui lorsqu'une arme se recharge
- Un ui qui montre le temps restant avant la fin de la wave en cours et à quelle numéro wave le joueur est
- Un écran de GameOver
- Des trainées de couleur derrière les balles
- Un shader qui fait flasher les ennemis en blanc lorsqu'ils reçoivent des dégâts
- Le joueur subit du recul en recevant des dégats
- Le joueur possède une accélération et décélération
- La crosshair bouge de façon fluide vers la position de la souris
- Dans la scene AnimsScene, les 1eres animations ont été rajoutées (elles ne sont pas dans la démo parce qu'on a pas encore toutes nos anims et celles présentes ne fonctionnent pas avec le système de déplaceent/rotation actuel)
